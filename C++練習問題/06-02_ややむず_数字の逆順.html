数字の逆順 ややむず

問題

正の整数 n を逆から読んだ数を出力してください。
例えば 123 なら 321 です。1200 のように末尾に 0 がある場合、逆から読むと 0021 ですが、数としては 21 になります。

入力 / 出力

整数 n が1つ与えられます。逆から読んだ数を1行で出力してください。

制約

  • 1 ≤ n ≤ 999999999

入出力例

入力出力
123321
120021
ヒント： 答え用の変数 r を 0 で用意し、r = r * 10 + n % 10; → n = n / 10; を n が 0 になるまで繰り返すと、逆順の数が組み立てられます。
（各桁の和（06-01）の「桁の取り出し」の応用です。）
