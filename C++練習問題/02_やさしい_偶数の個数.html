偶数の個数 やさしい

問題

整数 N が与えられ、続けて N 個の整数が与えられます。そのうち偶数（2で割り切れる数）の個数を出力してください。0 や負の数も対象です。

入力

N
a1 a2 … aN

出力

偶数の個数を1行で出力してください。

制約

  • 1 ≤ N ≤ 100
  • 各整数は −1000 〜 1000

入出力例

入力出力
5
1 2 3 4 5		2
3
2 4 6		3
ヒント： x % 2 == 0 なら偶数です。0 も負の偶数も x % 2 == 0 で正しく判定できます。数えるための変数を 0 から始め、偶数のたびに +1 しましょう。
