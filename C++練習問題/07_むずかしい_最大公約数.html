最大公約数（ユークリッドの互除法） むずかしい

問題

2つの正の整数 a, b の最大公約数（GCD）を出力してください。

古代ギリシャから伝わるユークリッドの互除法を使うと簡単に求められます：

  • a を b で割った余りを r とする
  • a ← b、b ← r と置きかえる
  • b が 0 になったときの a が答え

入力 / 出力

1行に a と b が空白区切りで与えられます。最大公約数を1行で出力してください。

制約

  • 1 ≤ a, b ≤ 1000000

入出力例

入力出力
12 186
17 51（互いに素）
ヒント： 
while (b != 0) {
    int r = a % b;
    a = b;
    b = r;
}
// ここで a が最大公約数
余りを使ってどんどん小さくしていくのがポイントです。
