素因数分解 ややむず

問題

整数 n を素数の掛け算に分解（素因数分解）し、素因数を小さい順に1行に1つずつ出力してください。
例：12 = 2 × 2 × 3 なので、2、2、3 を順に出力します。同じ素数が複数回あれば、その回数だけ出力します。

入力 / 出力

整数 n が1つ与えられます。素因数を小さい順に1行ずつ出力してください。

制約

  • 2 ≤ n ≤ 1000000

入出力例

入力出力
122
2
3
77
ヒント： i = 2 から順に「n が i で割り切れる間、i を出力して n を i で割る」を繰り返します（while を for の中に入れる）。
i は i * i <= n の間だけ調べれば十分。ループが終わったとき n が 1 より大きければ、残った n 自身が素数なので最後に出力します。
