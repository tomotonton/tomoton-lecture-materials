階乗 ややむず

問題

整数 n の階乗 n! を出力してください。
n! = 1 × 2 × 3 × … × n です。例えば 5! = 1×2×3×4×5 = 120。
ただし 0! = 1 と決められています。

入力 / 出力

整数 n が1つ与えられます。n! を1行で出力してください。

制約

  • 0 ≤ n ≤ 12（13以上は int の範囲を超えます）

入出力例

入力出力
5120
01
ヒント： 答え用の変数を 1 で初期化し（0 ではダメ！）、for で 1 から n まで順に掛けていきます。
n = 0 のときはループが1回も回らないので、初期値の 1 がそのまま答えになります。
