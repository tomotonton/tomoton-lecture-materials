回文数 ややむず

問題

「しんぶんし」のように逆から読んでも同じになるものを回文といいます。
正の整数 n が、逆から読んでも同じ数（回文数）なら YES、違えば NO を出力してください。
例：121 や 1221 は回文数、123 は回文数ではありません。

入力 / 出力

整数 n が1つ与えられます。YES または NO を1行で出力してください。

制約

  • 1 ≤ n ≤ 999999999

入出力例

入力出力
121YES
123NO
ヒント： 「数字の逆順（06-02）」の方法で逆順の数を作り、元の数と等しいかを比べます。
逆順を作ると元の変数は壊れてしまうので、先に元の値を別の変数にコピーしておくのがポイントです。
