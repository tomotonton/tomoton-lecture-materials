温度変換（摂氏 → 華氏） 入門

問題

摂氏温度 C（整数）が与えられます。華氏温度 F に変換して出力してください。変換式は次のとおりです。

F = C × 9 / 5 + 32

入力 / 出力

整数 C が1つ与えられます。華氏温度を1行で出力してください（小数になる場合もあります）。

入出力例

入力(℃)出力(℉)
032
100212
3798.6
⚠ 落とし穴： C * 9 / 5 + 32 をすべて整数で計算すると、9 / 5 が整数の割り算で 1 になってしまい（本当は 1.8）、答えがずれます。
型変換で解決： 式の中に小数（実数）を混ぜると、実数の計算になります。
C * 9.0 / 5.0 + 32 のように 9.0, 5.0 と書く（または (double)C を使う）と正しく計算できます。
