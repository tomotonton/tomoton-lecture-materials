整数の合計 やさしい

「ひとつ以上の入力を読み取り → ロジックを組んで → 出力する」練習です。下のエディタに直接書いて、実行で試し、提出でテストを通しましょう。

問題

整数 N が与えられ、続けて N 個の整数が与えられます。それら N 個の整数の合計を出力してください。

入力

N
a1 a2 … aN

1行目に個数 N、2行目に N 個の整数が空白区切りで与えられます（改行区切りでも構いません）。

出力

合計値を1行で出力してください。

制約

  • 1 ≤ N ≤ 100
  • 各整数は −1000 〜 1000

入出力例

入力出力
3
10 20 30		60
1
5		5
考え方のヒント： 合計を入れる変数 sum を 0 で用意し、for ループで1個ずつ cin >> x; して sum に足していきます。全部読んだら sum を出力します。
（配列はまだ使いません。「読みながら足す」だけで解けます。）
