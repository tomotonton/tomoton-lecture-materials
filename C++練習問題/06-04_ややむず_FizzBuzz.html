FizzBuzz ややむず

問題

世界中の面接で出題されてきた超有名問題です。1 から n までの整数を順に1行ずつ出力してください。ただし、

  • 3の倍数のときは数の代わりに Fizz
  • 5の倍数のときは数の代わりに Buzz
  • 3と5両方の倍数（15の倍数）のときは FizzBuzz

と出力します。

入力 / 出力

整数 n が1つ与えられます。1 から n までを上のルールで1行ずつ出力してください。

制約

  • 1 ≤ n ≤ 100

入出力例

入力出力
51
2
Fizz
4
Buzz
ヒント： for ループの中で「15の倍数 → 3の倍数 → 5の倍数 → それ以外」の順に if / else if で判定します。
先に15の倍数を調べるのがコツ（3の倍数を先に調べると、15 が Fizz になってしまう）。
