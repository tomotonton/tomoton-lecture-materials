割り算を小数で 入門

問題

2つの整数 a, b が与えられます。a ÷ b の結果を小数（実数）で出力してください。

入力 / 出力

1行に a と b が与えられます。割り算の結果を1行で出力してください。

入出力例

入力出力
7 23.5
1 40.25
6 23
⚠ よくある間違い： int a, b; のまま cout << a / b; と書くと、整数どうしの割り算になって小数点以下が切り捨てられます（7 / 2 が 3 になってしまう）。
型変換で解決： どちらか一方を double（実数）に変換すれば、小数の割り算になります。
cout << (double)a / b;  …  a を実数に変換してから割る → 3.5
これをキャスト（型変換）と呼びます。
