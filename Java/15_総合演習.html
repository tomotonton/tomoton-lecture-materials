■ 問3・問4：プログラムトレース練習

各問題を自分で解いてから答えを確認しましょう。

【第67回 問3(17)】単項演算子

int a = 6;
System.out.println(-a - 2);
答えを見る
-8

単項の -a が先に評価されて -6。その後 -6 - 2 = -8。

【第67回 問3(18)】型変換のコンパイルエラー

int x = 35;
short y = x + 10000;
System.out.println(y);
答えを見る
コンパイルエラー（4行目）

x + 10000 の結果は int 型。short への代入にはキャストが必要。

【第67回 問3(20)】整数除算

int a = 3 / 8;
double b = 3 / 8.0;
System.out.println(a + " " + b);
答えを見る
0 0.375

3/8 は整数除算で 0。3/8.0 は double 計算で 0.375。

【第67回 問4(22)】配列とfor文

int[] data = {3, 5, 6, 9, 10, 13, 16};
for (int i = 0; i < data.length; i += 2) {
    System.out.print(data[i] + ", ");
}
答えを見る
3, 6, 10, 16,

i = 0, 2, 4, 6 → data[0]=3, data[2]=6, data[4]=10, data[6]=16

【第67回 問4(23)】switchのフォールスルー

int a = 4;
switch (a - a / 2 * 2) {
    case 0:
        System.out.println("zero");
    case 1:
        System.out.println("one");
}
答えを見る
zero one

a - a/2*2 = 4 - 2*2 = 0 → case 0 に入る。break なしでフォールスルーして case 1 も実行。

【第67回 問4(24)】前置デクリメントとwhile

int a = 5;
while (a > 0) {
    System.out.println(--a);
    a--;
}
答えを見る
4 2 0

1ループで a が2減ります。前置 --a で表示前に減算。

【第63回 問3(18)】キャスト

int t = (int)4.99999;
System.out.print(t);
答えを見る
4

double → int のキャストは小数点以下を切り捨て。四捨五入ではありません。

【第63回 問4(22)】String の for ループ

    mozi = mozi + "@";

【第67回 問5 空欄 (30)】rankの初期値

「100mのタイムが速い順に5人格納。新しい走者の順位を求める」プログラム。

int rank = [ (30) ]; の空欄を選べ。

選択肢：ア -1　イ 0　ウ 1　エ 5

答えを見る

答え：ウ 1

新しい走者は最初「1位」と仮定して、比較しながら順位を下げていく設計。rank の初期値は 1。

【第67回 問6 空欄 (35)】バス停名の存在チェック

バスの運賃計算プログラム。存在しないバス停名が入力された場合にエラーを表示する。

if ( [ (35) ] ) { System.out.println("Error"); return; }

選択肢：ア f == -1 && t == -1　イ f == -1 || t == -1　ウ f >= 0 && t >= 0　エ f >= 0 || t >= 0

答えを見る

答え：イ f == -1 || t == -1

indexOf が -1 を返す = 存在しないバス停。乗車か降車のどちらか一方でも -1 ならエラー → ||（OR）を使います。

問題15-2

次のJavaコードの出力として正しいものはどれか。

int t = (int) 4.99999;
System.out.print(t);
解説を表示 正解：イ
double から int へのキャストは小数点以下を切り捨て。四捨五入ではないので 4.99999 → 4。Math.round() を使えば四捨五入できる。第63回 問3(18) と同じパターン。

問題15-3

次のJavaコードの出力として正しいものはどれか。

int a = 5;
while (a > 0) {
    System.out.println(--a);
    a--;
}
解説を表示 正解：ア
前置 --a で表示前に a を減算、さらに a-- で 1 減算。1ループで a が 2 減る。トレース：a=5→--a→4 表示→a--→3、a=3→--a→2 表示→a--→1、a=1→--a→0 表示→a--→-1、a=-1 で a > 0 が false。第67回 問4(24) と同じパターン。

問題15-4

バス停名のリスト busStopName = "ABCDEFG" に対し、int f = busStopName.indexOf(from);, int t = busStopName.indexOf(to); としたとき、「乗車バス停 from または降車バス停 to のどちらか一方でも存在しなければエラー」を判定する条件式として正しいものはどれか。

解説を表示 正解：イ
indexOf は見つからないと -1 を返す。「どちらか一方でも -1 ならエラー」なので OR（||）で連結する。&&（AND）だと「両方が -1 のときだけ」になってしまう。第67回 問6 空欄(35) と同じパターン。

問題15-5

コマンドラインから受け取った最初の1個の文字列を実数値（double）として解釈する正しい書き方はどれか。

解説を表示 正解：ア
コマンドライン引数は args[0] から始まる（C言語と違って実行コマンド名は含まれない）。実数値なので Double.parseDouble を使う。Integer.parseInt は int 用なので小数点付きの文字列を渡すと NumberFormatException が出る。第67回 問5(29) と同じパターン。
