■ フローチャート（SVGダイアグラム）の編集ルール
====================================================

【ルール1】True（条件成立）の矢印は必ず下向き
  - 条件ひし形からTrueの場合は「下」へ矢印を引く
  - Falseの場合は横（左or右）へ引く
  - 例：while文 → True=下向きでループ処理へ、False=横向きで終了へ
  - 例：if文    → True=下向きで処理へ、False=横向きでスキップまたはelse処理へ

【ルール2】True / False で統一（ファイル全体）
  - フローチャートのラベル・本文テキスト・コードコメントを含む全箇所で「真」「偽」は使用しない
  - すべて True / False に統一する
    例：「// score=75 → 90以上は False → 70以上は True → grade = "B"」

■ quiz-firebase.js の仕様・変更履歴
====================================================

【仕様概要】
- Firebase Realtime Database を使った学生回答集計システム
- 回答はlocalStorageに保存（タイムスタンプ付き）
- 棒グラフはChart.jsで描画（過去48時間集計）

【リセット機能の仕組み】
- 先生が「管理者：データリセット」ボタン→パスワード「ignas」で実行
- Firebase の meta/{qid}/resetAt にリセット時刻を書き込む
- 学生側はリアルタイム監視（onValue）で即座に検知 → 再回答可能になる
- ページを開いたままでもリセットが即反映される（2026-05-15 修正）

【修正履歴】
2026-05-15: initQuiz に onValue(meta/{qid}/resetAt) を追加。
  先生のリセット後、学生がページを開いたままでも即座に再回答可能になった。
  （従来はページリロードしないと反映されなかった）
