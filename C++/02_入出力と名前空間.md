## 📘 C++とCの違い（入出力・名前空間）

- Cでは `printf`, `scanf` を使うが、C++では `cout`, `cin` を使う
- C++の入出力は演算子 `<<` や `>>` を用いて直感的に書ける
- `std::cout` や `std::cin` のように、標準ライブラリの機能は `std` 名前空間に含まれている

---

## 💡 入出力と演算子 `<<`, `>>`

### 入出力の基本
C++ では、以下の2つを使って入出力を行います：

- `cout`：標準出力（画面）に文字や数値を表示する
- `cin`：標準入力（キーボード）から値を受け取る

### `<<` と `>>` の意味

- `<<` は「出力ストリーム演算子」
  - `cout` で使うと、「右側の値を左側の出力ストリーム（画面）に送る」
  - 例：`cout << "Hello";` → "Hello" が画面に出力される

- `>>` は「入力ストリーム演算子」
  - `cin` で使うと、「左側の入力ストリーム（キーボード）から右側の変数に値を読み込む」
  - 例：`cin >> age;` → キーボードから入力された値を `age` に格納する

💡 向きで覚えると簡単：
- `<<`：左に流す → 出力
- `>>`：右に流す → 入力

```cpp
cout << "名前を入力してください："; // 出力
cin >> name;                         // 入力
```

※ `<<` や `>>` は演算子オーバーロードにより、通常のビット演算とは異なる意味で使われています。

## 💻 実践ステップ

### 02_01_output.cpp（出力）
```cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, C++ World!" << endl;
    return 0;
}
```

---

### 02_02_input.cpp（入力）
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    string name;
    cout << "名前を入力してください：";
    cin >> name;
    cout << "こんにちは、" << name << " さん！" << endl;
    return 0;
}
```

---

### 02_03_namespace.cpp（namespaceの省略）
```cpp
#include <iostream>

int main() {
    std::cout << "名前空間stdを省略せずに使う例です。" << std::endl;
    return 0;
}
```
