## 📘 C++のデータ型とCとの違い

C++ で使う主なデータ型は以下のとおりです。C言語と同様に `int`、`double`、`char` などがありますが、C++ では次のような点が異なります：

- `bool` 型が標準装備（Cでは `_Bool` や `<stdbool.h>` が必要）
- `string` 型が使える（Cでは `char[]` や `char*` で手動管理）
- 出力に `cout` を使うことで構文が直感的になる

Cでは少し面倒だった文字列や真偽値の扱いが、C++ ではシンプルに書けます。

| 型名     | 意味             | 例               |
|----------|------------------|------------------|
| `int`    | 整数             | 1, -10, 250       |
| `double` | 実数（小数）     | 3.14, -0.5       |
| `char`   | 文字1つ          | 'A', 'z'         |
| `bool`   | 真偽値           | true, false      |
| `string` | 文字列           | "Hello", "ABC"   |

※ `string` を使うには `<string>` をインクルードしてください。

## 💻 実践ステップ

### 03_01_var_int_string.cpp（intとstring）
```cpp
#include <iostream>
#include <string>
using namespace std;

int main() {
    int score = 85;
    string name = "田中";

    cout << name << " さんの点数は " << score << " 点です。" << endl;
    return 0;
}
```

### 03_02_var_double_char.cpp（doubleとchar）
```cpp
#include <iostream>
using namespace std;

int main() {
    double pi = 3.1415;
    char grade = 'A';

    cout << "円周率: " << pi << endl;
    cout << "評価: " << grade << endl;
    return 0;
}
```

---

### 03_03_var_bool.cpp（boolとboolalpha）
```cpp
#include <iostream>
using namespace std;

int main() {
    bool isClear = true;
    bool isGameOver = false;

    cout << "クリア状況（数値）: " << isClear << endl;
    cout << "ゲームオーバー（数値）: " << isGameOver << endl;

    // 文字で表示する場合
    cout << boolalpha; // true/false を文字で表示
    cout << "クリア状況（文字）: " << isClear << endl;
    cout << "ゲームオーバー（文字）: " << isGameOver << endl;

    return 0;
}
```

💡 `bool` は通常 1/0 で表示されますが、`boolalpha` を使うと true/false の文字として出力されます。

> `boolalpha` は「ストリームマニピュレータ」と呼ばれる特殊な命令で、`cout` の出力形式を一時的に変更します。
