---

## 📘 C++とCの違い（演算と条件分岐）

C++ とCで、演算子や条件分岐の基本構造は同じですが、次のような違いがあります：

- `bool` 型がある（Cでは0と1で表現）
- 真偽値が `true` / `false` で明示できる（Cでは `1` / `0`）
- `string` や `cout` と組み合わせて、自然な表現が可能

---

## 💻 実践ステップ

### 04_01_arithmetic.cpp（算術演算子）
```cpp
#include <iostream>
using namespace std;

int main() {
    int a = 10;
    int b = 3;

    cout << "a + b = " << a + b << endl;
    cout << "a - b = " << a - b << endl;
    cout << "a * b = " << a * b << endl;
    cout << "a / b = " << a / b << endl;
    cout << "a % b = " << a % b << endl;
    return 0;
}
```

---

### 04_02_comparison.cpp（比較演算子）
```cpp
#include <iostream>
using namespace std;

int main() {
    int score = 75;
    cout << "合格か？：" << (score >= 60) << endl; // true = 1
    return 0;
}
```

---

### 04_03_if_statement.cpp（if文の基本）
```cpp
#include <iostream>
using namespace std;

int main() {
    int score;
    cout << "スコアを入力してください：";
    cin >> score;

    if (score >= 60) {
        cout << "合格です。" << endl;
    } else {
        cout << "不合格です。" << endl;
    }
    return 0;
}
```

---

### 04_04_if_bool.cpp（bool型とifの組み合わせ）
```cpp
#include <iostream>
using namespace std;

int main() {
    bool isClear = true;

    if (isClear) {
        cout << "ステージクリア！" << endl;
    } else {
        cout << "やり直し！" << endl;
    }
    return 0;
}
```

> C++ では `bool` 型を直接 `if` に使えるため、Cよりもわかりやすく書けます。

## 🔍 演習

1. 2つの整数を入力し、大きい方を表示するプログラムを作成してください（04_05_max.cpp）
2. 点数を入力して、80点以上なら「優」、60点以上なら「良」、40点以上なら「可」、それ未満なら「不可」と表示してください（04_06_grade.cpp）

```cpp
#include <iostream>
using namespace std;

int main() {
    bool isLoggedIn = false; // false = 未ログイン

    if (isLoggedIn) {
        cout << "ようこそ、ユーザーさん！" << endl;
    } else {
        cout << "ログインしてください。" << endl;
    }
    return 0;
}
```
