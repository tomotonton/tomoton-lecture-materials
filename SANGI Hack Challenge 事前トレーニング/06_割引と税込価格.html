割引と税込価格 LV2 ・ 20点 👑

問題

ある商品の本体価格 P 円、割引率 D パーセント、消費税率 T パーセントが与えられます。次の順で計算した支払い金額（税込）を求めてください。

  1. 割引後の価格 = P から D パーセントを割り引いた金額。小数点以下は切り捨てる。
  2. 税込価格 = 割引後の価格に T パーセントの税を加えた金額。小数点以下は切り捨てる。

例えば P = 1000, D = 20, T = 10 のとき、割引後は 1000 × 0.8 = 800 円、税込は 800 × 1.1 = 880 円です。

制約

  • 入力はすべて整数です。
  • 1 <= P <= 1000000
  • 0 <= D <= 100
  • 0 <= T <= 100

入力

入力は以下の形式で標準入力から与えられます。

P D T

出力

税込価格を半角数字で 1 行に出力してください。

入出力例

入力出力
1000 20 10
880
298 0 10
327

💡 学習アドバイス

実務でもよく出る「割合の計算」を、小数を使わず整数だけで正確に行う練習です。ポイントは 2 つ。

  • 計算の順番: 必ず「先に割引 → そのあと税」。順番を変えると答えが変わります。
  • 切り捨て: 整数どうしの割り算（/）は、C / C++ / Java では自動的に小数点以下が切り捨てられます。これを利用して、金額 * (100 - D) / 100 のように計算します。

100 分の何 を整数の掛け算と割り算で表すコツがつかめると、消費税や割引の計算が怖くなくなります。問題文の例で、自分の計算が 880 になるか確かめてから提出しましょう。
