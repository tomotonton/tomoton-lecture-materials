C言語を使う準備

1. フォルダーを作る

デスクトップなど、自分で分かる場所に新しくフォルダーを作る
名前は C言語 にする

2. Visual Studio（VS2022）をインストールする

次のサイトから、Visual Studio 2022 Community をインストールする
インストール時に「C++によるデスクトップ開発」を選ぶ

https://visualstudio.microsoft.com/ja/vs/community/

3. コマンドプロンプトのショートカットをコピーする

  1. スタートメニューから
    Developer Command Prompt for VS 2022 を探す
  2. 右クリックして「ファイルの場所を開く」を選ぶ
  3. 出てきたショートカットを コピー して、
    C言語フォルダに貼り付ける

4. VSCode（Visual Studio Code）をインストールする

次のサイトから VSCode をインストールする

https://code.visualstudio.com/

5. VSCodeの設定（.cファイルだけShift-JISで開くようにする）

  1. VSCodeを起動する
  2. 左下の「歯車マーク（設定）」をクリックして「設定」を選ぶ

  3. 右上の「{}（設定ファイルを開く）」をクリックして settings.json を開く

  4. 以下のように書き加える：

"[c]": {
    "files.encoding": "shiftjis"
},
"files.autoGuessEncoding": true

これで .c ファイルのときだけ、文字コードが Shift-JIS になる
他のファイルには影響しない設定になる

実行方法（プログラムを書いて動かす）

1. コマンドプロンプトを起動する

C言語フォルダにコピーした
Developer Command Prompt for VS 2022 のショートカットをダブルクリックして開く

2. VSCodeを起動する

次のように入力してエンターキーを押す

>code

VSCodeが起動する

3. フォルダーを開く

VSCodeの左上「ファイル」から「フォルダーを開く」を選び、
C言語フォルダを開く

4. 新しいファイルを作成する

ファイル名：01_helloWorld.c
新規ファイルとして作成し、中身を次のように書く

#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}

5. 実行する

画面右上にある 三角のボタン（再生マーク） をクリックする
少し待つと、下に実行結果が表示される

おわりに

この手順でC言語の準備が完了しました。
次の授業では、実際にいろんなプログラムを書いて学んでいきます。
