C言語 プログラムの作成・コンパイル・実行手順

① 作業フォルダを用意する

  1. デスクトップなど分かりやすい場所に C_lang というフォルダを作ります。
  2. その中に src フォルダ（ソースファイル保存用）を作ります。

② VSCodeポータブル環境をUSBメモリなどに展開する

  1. 用意された「VSCodeとGCC付きのUSBパック」をそのままフォルダにコピーします。
  2. VSCodeポータブル.exe をダブルクリックして起動します。

③ プログラムファイル（例：hello.c）を作成する

  1. VSCode上で src フォルダを開き、新規ファイルを作成します。
  2. ファイル名は hello.c など、拡張子 .c にします。
  3. 次のようなC言語のプログラムを書きます：
#include <stdio.h>

int main(void) {
    printf("Hello, World!\n");
    return 0;
}

④ ターミナルでコンパイルして実行する

  1. VSCodeの下にある「ターミナル」タブを開きます。
  2. 次のコマンドを入力して実行します：
../GCC/bin/gcc hello.c -o hello.exe
  • ※ ../GCC/bin/gcc はUSBメモリ上にあるGCCコンパイラのパスです。
  1. 次に、生成された実行ファイルを次のようにして実行します：
./hello.exe

⑤ 実行結果を確認する

「Hello, World!」と表示されれば成功です 🎉

補足

  • 文字化けを防ぐため、拡張子が .c のファイルは Shift-JIS で保存される設定になっています。
  • 困ったことがあれば、担当教員に聞いてください。
