N個の整数の合計 入門

入力が2行になります。「まず個数、つぎにその個数ぶんのデータ」という、よくある入力パターンです。

問題

1行目に整数の個数 N、2行目に N 個の整数が与えられます。それら N 個の合計を出力してください。

入力

N
a1 a2 … aN

入出力例

入力出力
3
10 20 30		60
1
5		5
💡 ヒント（クリックで必要な知識が開きます）
  • まず個数を読む： int n; scanf("%d", &n);
  • 合計を入れる変数を 0 で用意： int sum = 0;
  • N回くり返して読みながら足す： 
    for (int i = 0; i < n; i++) {
    int x;
    scanf("%d", &x);
    sum += x;          /* sum = sum + x; と同じ */
}
  • 最後に出力： printf("%d\n", sum);
  • ※ 配列はまだ使いません。「1個読んでは足す」をくり返すだけで解けます。
