最大値 やさしい

問題

整数の個数 N、続けて N 個の整数が与えられます。その中で最大の値を出力してください。
（配列はまだ使いません。「読みながら、今までの最大より大きければ更新」で解けます。）

入力

N
a1 a2 … aN

入出力例

入力出力
5
3 1 4 1 5		5
4
-5 -2 -9 -1		-1
💡 ヒント（クリックで必要な知識が開きます）
  • 最初に読んだ数を「仮の最大」にして、以降は「今の最大より大きければ更新」します。
  • ⚠ 仮の最大を 0 で始めると、全部が負の数のとき答えが 0 になって間違えます。最初の値で初期化しましょう。
  • 例： if (i == 0 || x > mx) mx = x;（i==0 のとき＝最初の値で必ず更新）
