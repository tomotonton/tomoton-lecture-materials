絶対値 やさしい

問題

整数を1つ読み取り、その絶対値（マイナスを取り除いた、0以上の大きさ）を出力してください。

入出力例

入力出力
55
-55
💡 ヒント（クリックで必要な知識が開きます）
  • 条件で分ける（if文）： 値が負のときだけ符号を反転します。
  • if (n < 0) {
        n = -n;
    }     としてから n を出力すれば絶対値になります。
  • （<math.h> の abs を使う方法もありますが、まずは if 文で考えてみましょう。）
