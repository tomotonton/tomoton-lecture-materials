最大公約数（ユークリッドの互除法） むずかしい

問題

2つの正の整数 a, b の最大公約数（GCD）を出力してください。
古代ギリシャから伝わるユークリッドの互除法を使うと簡単に求められます。

入出力例

入力出力
12 186
17 51（互いに素）
💡 ヒント（クリックで必要な知識が開きます）
  • ユークリッドの互除法： a を b で割った余りを r とし、a ← b, b ← r をくり返す。b が 0 になったときの a が答え。
  • while (b != 0) {
    int r = a % b;
    a = b;
    b = r;
}
printf("%d\n", a);
