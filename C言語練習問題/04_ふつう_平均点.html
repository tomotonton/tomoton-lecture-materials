平均点 ふつう

問題

N 人分の点数（整数）が与えられます。その平均点を、小数第1位まで出力してください。

入力

N
p1 p2 … pN

入出力例

入力出力
3
50 60 70		60.0
2
59 60		59.5
💡 ヒント（クリックで必要な知識が開きます）
  • まず合計 sum を求めます（int sum = 0; からループで sum += p;）。
  • ⚠ sum / n を整数のまま割ると小数点以下が切り捨てられます。(double) でキャストして割りましょう： (double)sum / n
  • 小数第1位まで出力： printf("%.1f\n", (double)sum / n);
