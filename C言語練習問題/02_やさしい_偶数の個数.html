偶数の個数 やさしい

問題

整数の個数 N、続けて N 個の整数が与えられます。そのうち偶数（2で割り切れる数）の個数を出力してください。0 や負の数も対象です。

入力

N
a1 a2 … aN

入出力例

入力出力
5
1 2 3 4 5		2
💡 ヒント（クリックで必要な知識が開きます）
  • 偶数の判定： x % 2 == 0 なら偶数（0 や負の偶数も正しく判定できます）。
  • 数える変数を 0 で用意： int count = 0;
  • N回くり返して、偶数なら +1： 
    for (int i = 0; i < n; i++) {
    int x;
    scanf("%d", &x);
    if (x % 2 == 0) count++;
}
